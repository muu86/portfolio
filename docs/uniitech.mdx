# Oracle 프로시저를 SpringBoot Batch로 이전 프로젝트

회사의 모든 **DB 프로시저**를 **SpringBoot Batch**로 이전하는 프로젝트에 참여했습니다. 모든 개발 인력이 참여한 대규모
프로젝트였기 때문에 제 기여도는 상대적으로 크지 않았지만, 이 프로젝트를 통해 회사에서 사용되는 DB 프로시저를 파악하고
**커머스 전반에 걸친 프로세스**를 학습할 수 있었습니다.

이 프로젝트는 기술적인 관점뿐만 아니라 회사의 전반적인 운영 흐름에 대한 중요한 이해를 얻을 수 있는 기회였습니다.

# 위메프 & 홈앤쇼핑 제휴 프로젝트

위메프(WeMakePrice)와 홈앤쇼핑(Home&Shopping) 간의 상품 및 주문 연동 프로젝트에 참여하여, 상품 연동 작업을 담당했습니다.

- **API 연동**: 제공된 API 명세에 따라 REST API를 사용하여 상품 데이터를 연동했습니다.
- **자동화된 데이터 업데이트**: **Spring Scheduler**를 활용해 주기적인 데이터베이스 업데이트를 설정, 실시간으로 정확한
  데이터를 유지했습니다.
- **오류 처리 및 재시도 로직**: 데이터베이스에서 DB Flag 값을 사용하여 데이터를 성공, 실패, 대기 상태로 구분하고, 실패한
  데이터는 자동으로 재시도하여 연동을 맞추는 시스템을 구현했습니다.

# 정기 배송 프로젝트 (건강 보조 식품)

회사의 첫 정기 배송 프로젝트에 참여하여, 건강 보조 식품의 정기 배송 시스템을 구축했습니다. 프로젝트의 주요 목표는 기존
상품과 정기 배송 상품을 구분하고, PG사에서 발급받은 **billing key**를 이용해 정기 결제를 구현하는 것이었습니다.

- 아쉽게도 갓 입사한 저에게 핵심 업무는 주어지지 않았지만, 대신 유저에게 건강 설문을 시행하고, 설문 결과에 따라 맞춤형
  건강 식품을 추천하는 시스템을 구축하는 업무를 맡았습니다.

- **백오피스 시스템 개발**: **Spring**, **Extjs**, **JQuery**를 활용하여 고객 설문을 생성하고 수정하는 프로그램을
  개발했습니다. 설문 답변마다 점수를 부여하고, 고객 설문 결과에 따라 적합한 건강 식품을 추천하는 로직을 구현했습니다.
